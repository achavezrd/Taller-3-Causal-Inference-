# Taller-3-Causal-Inference-
Bases de datos para el taller 3


Para el primer problema, sleep_score.dta las variables son las siguientes:

-Atardecer: hora promedio del atardecer durante la semana del experimento.
-Sueño: horas de sueño promedio durante la semana del experimento.
-Puntaje: puntaje expresado en desviaciones estándar.
-Error: error promedio en la cantidad de sueño.

Para el tercer problema, utilicen la base de fabregas_data.dta y consulten el variables data RDD.txt para revisar las variables.
